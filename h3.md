https://httpd.apache.org/docs/2.4/vhosts/name-based.html <br>  
- Name-based Virtual Hostsin avulla voidaan ylläpitää yhden IP-osoitteen alla useita eri verkkosivuja.
- Palvelin käyttää käyttäjän selaimen lähettämää HTTP Host headeria päätelläkseen, mikä sivusto näytetään 

https://terokarvinen.com/2018/04/10/name-based-virtual-hosts-on-apache-multiple-websites-to-single-ip-address/ 
Tässä artikkelissa opastetaan kuinka pystytetään useampi sivusto yhdelle palvelimelle. Käydään läpi: 
- Apachen asennus 
- Virtuaalipalvelimen luonti 
- Name Based Virtual Host asetuksien määrittely 
- HTML sivujen luonti 
- Testaus <code>curl</code> komennolla
